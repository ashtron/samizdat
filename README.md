samizdat is a web app for **privately** tracking the media you read, watch, and listen to.

### Demo

https://github.com/ashtron/samizdat/assets/2673057/0ef33221-6917-4f74-9088-f6a4f05ccc51

### Technologies

* [NextJS](https://github.com/vercel/next.js/)
* [Supabase](https://github.com/supabase)
* [Pico CSS](https://github.com/picocss/pico)

### APIs

* [Open Library](https://openlibrary.org/)
* [TMDB](https://www.themoviedb.org/)
* [Discogs](https://www.discogs.com/)
